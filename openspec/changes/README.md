# OpenSpec Changes

Create one folder per proposed change:

```text
openspec/changes/{change-id}/
  proposal.md
  tasks.md
  design.md        # optional
```

Use verb-led change IDs:

```text
add-locust-metrics-collector
add-artifact-repository-oss
change-lane-namespace-strategy
```

After a change is accepted and implemented, merge the final capability requirements into `openspec/specs/*/spec.md`.
